펀딩 관련 안내

1. message 부분 푸시 알림 완료

2. user부분에 firebase message token, push token 추가했으니 최하단 확인, 사용자 프로필 가져오는 부분에서 message token 추가

3. 이미지 전부 string형식 url로 변경 완료

4. 카페 찜 true false 제공. 카페 리스트랑 상세정보 부분에 user_id request 추가되었음

5. 카페 정렬 기준 수정

6. 게임, 카페 이미지가 잘 작동되지 않는 현상 해결

7. 게임 부분에도 자신이 위시리스트에 추가 했는지 안했는지 표시. url에 user_id 추가됨

8. 채팅방 관련 부분 제작 완료

9. 모든 HTTP method를 GET 방식에서 POST방식으로 교체 완료

10. 커뮤니티 글 임시저장 제작 완료. 임시저장은 기존 커뮤니티 글 업로드에서 함

11. 커뮤니티 글, 댓글 신고 제작 완료

12. user부분에 친구 관련 기능 제작 완료
