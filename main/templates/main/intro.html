사용 방법 안내


현재 링크 : https://playdeserver-owoyh.run.goorm.io/
현재 링크 뒤에 작동 방법을 더해 사용


이미지 가져오는 방법 : 서버 링크 + 이미지 경로
작동 방법 안에 쓰여져있는 한글은 변수를 입력하는 칸
리턴 정보중 소괄호 안에 있는 정보들은 여러개가 주어질 수 있음
id는 사용자에게 보여지면 안됨

게임 리스트 표시를 위한 게임 정보
작동 방법 : game_list
리턴 정보 : (게임id, 게임 이름, 게임사진 작은버전)
게임은 br태그로 구분되어 있으며 게임 정보는 p태그로 구분되어서 제공됨. 인기 많은순으로 제공되며 최대 100개를 리턴함

게임 세부 정보
작동 방법 : game/게임 id
리턴 정보 : 게임id, 게임 이름, 게임 이용 가능 최소 인원수, 게임 이용 가능 최대 인원수, 게임 소개글, 게임사진 큰버전
해당 정보가 p태그로 구분되어서 제공됨

카페 이름, 위치정보
작동 방법 : cafelatlng/사용자위도/사용자경도
리턴 정보 : (카페 id, 카페 이름, 카페 위도, 카페 경도)
해당 정보가 br태그와 p태그로 구분되어 있으며, 사용자에게 2km 내에 있는 카페를 가장 가까운 순으로 최대 5개를 반환함

위시리스트 정보
작동 방법 : get_wishlist/사용자토큰
리턴 정보 : (게임id, 게임 이름, 게임사진 작은버전)
게임은 br태그로 구분되어 있으며 게임 정보는 p태그로 구분되어서 제공됨.

위시리스트 추가
작동 방법 : add_wishlist/사용자토큰/게임id
리턴 정보 : SUCCESS, FAIL
사용자의 위시리스트에 게임을 추가함

위시리스트 삭제
작동 방법 : del_wishlist/사용자토큰/게임id
리턴 정보 : SUCCESS, FAIL
해당 게임을 사용자의 위시리스트에서 삭제함

차단목록
작동 방법 : get_block/사용자토큰
리턴 정보 : (상대유저id, 상대방 닉네임, 상대 프사 작은버전)
사용자의 차단 목록 관리를 위한 정보임

상대방 차단하기
작동 방법 : add_block/사용자토큰/상대방id
리턴 정보 : SUCCESS, FAIL
상대방을 차단 목록에 추가해서 서로 만나요에서 보이지 않고 상대방이 채팅을 보내도 보이지 않음

차단 해제
작동 방법 : del_block/사용자토큰/상대방id
리턴 정보 : SUCCESS, FAIL
상대방을 차단 목록에서 삭제해서 다시 만나요에서 보거나 채팅을 칠 수 있음

저장한 카페 목록
작동 방법 : get_fav_cafe/사용자토큰
리턴 정보 : (카페id, 카페이름, 카페 사진 작은버전)

저장한 카페 추가
작동 방법 : add_fav_cafe/사용자토큰/카페id
리턴 정보 : SUCCESS, FAIL

저장한 카페 삭제
작동 방법 : del_fav_cafe/사용자토큰/카페id
리턴 정보 : SUCCESS, FAIL

유저 평가 추가, 수정
작동 방법 : add_usercomment/사용자토큰/상대방id/점수/내용
사용자가 상대방을 처음 평가할 경우 새로 생기고 이미 존재할 경우 수정해서 데이터베이스에 저장함.

유저 평가 불러오기
작동 방법 : get_usercomment/사용자토큰||상대방id/사용자토큰일경우: 1, 상대방id일경우: 0
리턴 정보 : 평점 평균, (작성자id, 작성자닉네임, 작성자프사, 평가점수, 평가내용)
