body

KEY VALUE
user_id {user_id
tag {tag}
range {start,end}
top_id {top_board_id (NULLABLE)}

응답

속성 데이터타입 설명
meta.count number 커뮤니티 글 수 (댓글 포함으로 인해 요청한 수보다 많을 수 있음)

meta.end number 끝점
community[].kind string 글일경우:board, 댓글일경우:reply
community[].id number 글 id
community[].content string 글 내용
community[].created_at string 글 작성 시간 (%Y.%m.%d %H:%M)
community[].parent_id number 해당 글이 댓글일 경우 부모 글 (댓글일경우에만 주어짐)
community[].user.id number 작성자 id
community[].user.nickname string 작성자 닉네임
community[].user.image string 작성자 프사 url
