body

KEY VALUE
platform {이메일:0, 카카오:1, 네이버:2}
token {token} (카카오나 네이버일때만 입력)
email {email} (이메일로 로그인일때만 입력)
password {password} (이메일로 로그인일때만 입력)

속성 데이터타입 설명
id number, bool 로그인 성공할경우 사용자id, 존재하지 않는 계정일경우 false, 비밀번호가 틀릴경우 true
access bool 로그인 성공할경우 true, 실패할경우 false

회원가입

작동 방법 : {{url}}/user/join

body

KEY VALUE
platform {이메일:0, 카카오:1, 네이버:2}
token {token} (카카오나 네이버일때만 입력)
email {email} (카카오나 네이버일때는 생략 가능)
password {password} (이메일일때만 입력)
name {user_name}
nickname {nickname}
phone {phone_number} (NULLABLE)
age {user_age} (NULLABLE)

응답

속성 데이터타입 설명
access bool 회원가입 성공할경우 true, 실패할경우 false
message number, string 회원가입 성공했을 경우 사용자id, 실패했을 때 이미 존재하는 사용자일경우 ALREADY USER, 이미 존재하는 닉네임일 경우 ALREADY NICKNAME
