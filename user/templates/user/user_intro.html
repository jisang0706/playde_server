로그인

작동 방법 : user/login?platform={이메일: 0, 카카오:1, 네이버:2}&token={token (카카오나 네이버일때만 입력)} &email={email (이메일로 로그인일때만 입력)}&password={password (이메일로 로그인일때만 입력)}

ex : 이메일인경우 : {{url}}/user/login?platform=0&email=kookmin@gmail.com&password=wowwowowowwo
네이버나 카카오인 경우 : {{url}}/login?platform=2&token=123456789

응답

속성 데이터타입 설명
id number, bool 로그인 성공할경우 사용자id, 존재하지 않는 계정일경우 false, 비밀번호가 틀릴경우 true
access bool 로그인 성공할경우 true, 실패할경우 false

회원가입

작동 방법 : user/join?platform={이메일: 0, 카카오:1, 네이버:2}&token={token (카카오나 네이버일때만 입력)} &email={email (이메일로 로그인일때만 입력)}&password={password (이메일로 로그인일때만 입력)}&name={name(공백가능)} &nickname={nickname(공백가능)}&phone={phone_number(공백가능)}&age={age(공백가능)}

ex : {{url}}/user/join?platform=1&token=123456789&name=쿠민

응답

속성 데이터타입 설명
access bool 회원가입 성공할경우 true, 실패할경우 false
message number, string 회원가입 성공했을 경우 사용자id, 실패했을 때 이미 존재하는 사용자일경우 ALREADY USER, 이미 존재하는 닉네임일 경우 ALREADY NICKNAME

닉네임변경

작동 방법 : user/nickname/set?user_id={user_id}&nickname={new_nickname}

ex : {{url}}/user/nickname/set?id=2&nickname=temp

응답

속성 데이터타입 설명
access bool 닉네임 변경 성공할경우 true, 실패할경우 false
message string 닉네임 변경 성공했을 경우 SUCCESS, 실패했을 때 이미 존재하는 닉네임일 경우 ALREADY NICKNAME

차단목록

작동 방법 : user/block?user_id={user_id}

ex : {{url}}/user/block?user_id=2

속성 데이터타입 설명
meta.count number 차단한 유저 수
user[].id number 차단한 사용자 id
user[].nickname string 차단한 사용자 닉네임
user[].profile string 차단한 사용자 프사 경로

상대방 차단하기

작동 방법 : user/block/add?user_id={user_id}&his_id={target_user_id}

ex : {{url}}/user/block/add?user_id=2&his_id=1

응답

속성 데이터타입 설명
act bool 성공할경우 true, 이미 차단한 사용자일경우 false

차단 해제

작동 방법 : user/block/delete?user_id={user_id}&his_id={target_user_id}

ex : {{url}}/user/block/delete?user_id=2&his_id=1

응답

속성 데이터타입 설명
act bool 성공할경우 true, 이미 차단 해제한 사용자일경우 false

유저 평가 불러오기

작동 방법 : user/comment?user_id={user_id}

ex : {{url}}/user/comment?user_id=2

응답

속성 데이터타입 설명
meta.count number 평가 수
average number 평균 점수
comment[].id number 평가글id
comment[].score number 평가 점수
comment[].content string 평가 내용
comment[].writter.id number 평가글 작성자 id
comment[].writter.nickname string 평가글 작성자 닉네임
comment[].writter.profile string 평가글 작성자 프사 위치

유저 평가 추가, 수정

작동 방법 : user/comment/add?user_id={writter_id}&his_id={target_user_id}&score={score in range(0, 11)}&content={content}

ex : {{url}}/user/comment/add?user_id=1&his_id=2&score=10&content=너무 좋아요

응답

속성 데이터타입 설명
act bool true, false

사용자 프로필

사용 방법 : user/profile?user_id={user_id}

ex : {{url}}/user/profile?user_id=1

응답

속성 데이터타입 설명
id number 사용자 id
nickname string 사용자 닉네임
profile string 사용자 프사 위치
score number 사용자 평균 평가 점수

사용자 프로필사진 변경

url로 요청할 경우 요청 방식 : GET

사용 방법 : user/profile/image/set?user_id={user_id}&image_url={image_url}

ex : {{url}}/user/profile/image/set?user_id=2&image_url=http://google.com/image/1

응답은 아래 표와 동일함

jpg, png로 요청할 경우 요청 방식 : POST

사용 방법 : user/profile/image/set | body(user_id={user_id}, image={profile_image})

ex : {{url}}/user/profile/image/set | body(user_id=1, image=프사.jpg)

응답

속성 데이터타입 설명
act bool 작동 여부
id number 사용자 id
image string 프로필사진 위치
