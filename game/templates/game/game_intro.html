ex : {{url}}/game/list?range=0,10

응답

속성 데이터타입 설명
meta.count number 게임 갯수
meta.start number 시작점
meta.end number 끝점
games[].id number 게임 id
games[].kor_name string 게임 이름
games[].profile_img string 게임 프로필사진 위치
games[].level number 게임 난이도
games[].interest number 게임 인기도
