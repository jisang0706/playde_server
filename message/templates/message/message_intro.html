메시지 전송 알림

작동 방법 : message/send?writer_id={writer_id}&target_id={target_user_id}&content={content}

ex : {{url}}/message/send?writer_id=1&target_id=2&content=이미지

존재하지 않는 토큰일경우 오류 발생

응답

속성 데이터타입 설명
response string firebase push response

채팅방 목록 불러오기

작동 방법 : message/room/list?user_id={user_id}

ex: {{url}}/message/room/list?user_id=2

응답

속성 데이터타입 설명
count number 채팅방 갯수
rooms[i].id number 채팅방 id
rooms[i].token string 채팅방 토큰
rooms[i].user_count number 채팅방 참여자 수 (자신 제외)
rooms[i].users[j].id number 채팅방 참여자 id
rooms[i].users[j].nickname string 채팅방 참여자 닉네임
rooms[i].users[j].image string 채팅방 참여자 프사 url

채팅방 불러오기

작동 방법 : message/room?user_id={user_id}&his_id={other_user_id}

ex: {{url}}/message/room?user_id=2&his_id=3

응답

속성 데이터타입 설명
id number 채팅방 id
token string 채팅방 토큰
user_count number 채팅방 참여자 수 (자신 제외)
users[i].id number 채팅방 참여자 id
users[i].nickname number 채팅방 참여자 닉네임
users[i].image string 채팅방 참여자 프사 url
