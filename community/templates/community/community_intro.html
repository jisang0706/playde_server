임시저장 글 목록 불러오기

작동 방법 : {{url}}/community/temp/get

body

KEY VALUE
user_id {user_id}
range {start},{end}

임시저장 글 상세보기는 기존 커뮤니티 글 불러오는 방식과 동일하게 가져오면 되고 임시저장 글을 가져온 후 삭제하거나 업로드, 다시 임시저장을 할 경우 업로드 후 기존 글을 삭제해야함

응답

속성 데이터타입 설명
meta.count number 임시저장 글 갯수
meta.start number 시작점
meta.end number 끝점
community[].id number 임시저장 글 id
community[].content string 내용
community[].tag string 게시판
