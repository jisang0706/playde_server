커뮤니티 글 목록 불러오기

작동 방법 : community/get?user_id={user_id}&range={start, end}&tag={tag (NULLABLE)}&top_id={top_board_id (NULLABLE)}

ex : {{url}}/community/get?user_id=1&range=1,11&tag=만나요&top_id=1

응답

속성 데이터타입 설명
meta.count number 글 갯수
meta.start number 시작점
mera.end number 끝점
community[].board.id number 커뮤니티 글 id
community[].board.content string 커뮤니티 글 내용
community[].board.like number 커뮤니티 글 좋아요 개수
community[].board.my_like bool 자신이 해당 글에 좋아요를 눌렀는지
community[].board.created_at date 커뮤니티 글 작성시간
community[].board.visit number 커뮤니티 글 방문 수
community[].board.tag string 커뮤니티 글 게시판
community[].board.comment_cnt number 커뮤니티 글 댓글 수
community[].writer.id number 글 작성자 id
community[].writer.nickname string 글 작성자 닉네임
community[].writer.profile string 글 작성자 프로필사진 위치

커뮤니티 글 업로드

작동 방법 : community/board/upload?user_id={user_id}&content={content}&tag={tag (공백 가능)}&images={image_urls (url이 쉼표로 구분되며 공백 가능)}

ex : {{url}}/community/board/upload?user_id=1&content=안녕하세요 반가워요&tag=만나요&images=https://google.com/wow.png,http://naver.com/good.png

응답

속성 데이터타입 설명
act bool true, false

작동 방법 : community/board/delete?user_id={user_id}&board_id={board_id}

ex : {{url}}/community/board/delete?user_id=1&board_id=1

응답

속성 데이터타입 설명
act bool true, false

커뮤니티 글 좋아요

작동 방법 : community/like?user_id={user_id}&board_id={board_id}

ex : {{url}}/community/like?user_id=1&board_id=1

응답

act bool 좋아요 눌렀을경우 true, 취소했을경우 false

커뮤니티 글 상세보기

작동 방법 : community/board/{board_id}?user_id={user_id}

ex : {{url}}/community/board/1?user_id=1

응답

속성 데이터타입 설명
board.id number 본문id
board.content string 본문 내용
board.like number 글 좋아요 수
board.my_like bool 내가 글 좋아요 눌렀는지
board.created_at date 글 작성시간
board.visit number 글 방문 수
board.tag string 글 게시판
board.comment_cnt number 댓글 수
board.images[].id number 이미지 id
board.images[].order number 이미지 순서
board.images[].image string 이미지 위치
writer.id number 글 작성자 id
writer.nickname string 글 작성자 닉네임
writer.profile string 글 작성자 프로필사진 위치
comment[].id number 댓글 id
comment[].content string 댓글 내용
comment[].created_at date 댓글 작성 시간
comment[].writer.id number 댓글 작성자 id
comment[].writer.nickname string 댓글 작성자 닉네임
comment[].writer.profile string 댓글 작성자 프로필사진 위치
comment[].reply boolean 대댓글이면 True, 그냥 댓글이면 False

ex : {{url}}/community/board/image/delete?user_id=2&board_id=1&image_id=3

응답

속성 데이터타입 설명
act bool true, false
