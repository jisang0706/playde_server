카페 리스트

작동 방법 : cafe/list?coords={user coords (NULLABLE)}&sort={찜많은순:0, 가까운순:1 (coords가 주어졌을때만 가능), 가격낮은순:2}&range={range}&cafe_name={search_cafename (NULLABLE)}

ex : {{url}}/cafe/list?coors=127.12123,12.134&sort=0&range=1,11

응답

속성 데이터타입 설명
meta.count number 불러온 카페 개수
meta.start number 시작점
meta.end number 끝점
cafe[].id number 카페 id
cafe[].name string 카페 이름
cafe[].address string 도로명주소
cafe[].coords string 위경도 format(latitude,longitude)
cafe[].profile string 카페 프사 위치
cafe[].table_cnt number 카페 테이블 개수
cafe[].open time 카페 당일 오픈 시간 format(HH:mm:ss)
cafe[].close time 카페 당일 클로즈 시간 format(HH:mm:ss)
cafe[].like number 카페 찜한사람 수

카페 상세

작동 방법 : cafe/{cafe_id}

ex : {{url}}/cafe/1

속성 데이터타입 설명
id number 카페 id
name string 카페 이름
address string 도로명주소
coords string 위경도 format(latitude,longitude)
table_cnt number 카페 테이블 개수
open time 카페 당일 오픈 시간 format(HH:mm:ss)
close time 카페 당일 클로즈 시간 format(HH:mm:ss)
like number 카페 찜한사람 수
image[] string 카페 사진들

찜한 카페 목록

작동 방법 : cafe/fav?user_id={user_id}

ex : {{url}}/cafe/fav?user_id=1

속성 데이터타입 설명
meta.count number 카페 개수
cafe[].id number 카페 id
cafe[].name string 카페 이름
cafe[].profile string 키페 프사 위치

찜 카페 추가

작동 방법 : cafe/fav/add?user_id={user_id}&cafe_id={cafe_id}

ex : {{url}}/cafe/fav/add?user_id=1&cafe_id=1

속성 데이터타입 설명
act bool true, false

찜 카페 삭제

작동 방법 : cafe/fav/delete?user_id={user_id}&cafe_id={cafe_id}

ex : {{url}}/cafe/fav/delete?user_id=1&cafe_id=1

속성 데이터타입 설명
act bool true, false
